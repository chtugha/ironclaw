This guide takes you from zero to a running IronClaw instance on your own machine in under 10 minutes. No cloud account, no API key, and no separate database server required.

---

## What you need

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| RAM | 8 GB | 16 GB |
| Disk space | 4 GB free | 20 GB free |
| OS | Windows 10, macOS 12, or any modern Linux | — |
| GPU | None (CPU works) | NVIDIA / AMD / Apple Silicon |

---

## Setting Up Your Agent

<Steps>

<Step title="Install Ollama and pull a model">

[Ollama](https://ollama.com) runs local LLMs on your machine. Install it from [ollama.com/download](https://ollama.com/download), then open a terminal and pull a model:

```bash
# Fast and lightweight — best starting point (2 GB download)
ollama pull llama3.2

# Verify Ollama is working
ollama run llama3.2 "Say hello"
```

> **Tip:** If you have a GPU with 8+ GB VRAM or 16+ GB RAM, try `qwen2.5:14b` for better results.

Ollama starts automatically in the background. You can verify it is running at [http://localhost:11434](http://localhost:11434).

</Step>

<Step title="Install IronClaw">

<Tabs>
  <Tab title="macOS / Linux">

  ```bash
  curl --proto '=https' --tlsv1.2 -LsSf \
    https://github.com/nearai/ironclaw/releases/latest/download/ironclaw-installer.sh | sh
  ```

  Restart your shell, then verify:

  ```bash
  ironclaw --version
  ```

  </Tab>

  <Tab title="macOS (Homebrew)">

  ```bash
  brew install ironclaw
  ironclaw --version
  ```

  </Tab>

  <Tab title="Windows">

  Download and run the [Windows Installer (.msi)](https://github.com/nearai/ironclaw/releases/latest/download/ironclaw-x86_64-pc-windows-msvc.msi).

  Or use PowerShell:

  ```powershell
  irm https://github.com/nearai/ironclaw/releases/latest/download/ironclaw-installer.ps1 | iex
  ironclaw --version
  ```

  </Tab>

  <Tab title="Build from source">

  Requires [Rust 1.92+](https://rustup.rs).

  ```bash
  git clone https://github.com/nearai/ironclaw.git
  cd ironclaw
  cargo build --release
  # Binary: ./target/release/ironclaw
  ```

  </Tab>
</Tabs>

<Accordion title="⚠️ Command not found after installing">

If `ironclaw` is not found after installation, the installer directory may not be in your `PATH`. Add it:

```bash
# Add to ~/.bashrc or ~/.zshrc
export PATH="$HOME/.cargo/bin:$PATH"
source ~/.bashrc   # or ~/.zshrc
```

</Accordion>

</Step>

<Step title="Start IronClaw for the first time">

```bash
IRONCLAW_PROFILE=local ironclaw
```

The `local` profile uses an embedded database — no PostgreSQL installation needed.

On the **first run**, IronClaw walks you through a short setup:

**1. Database backend**

```
Select database backend:
  [1] libSQL (embedded, zero-config) ← press Enter to choose this
  [2] PostgreSQL
```

Press `1` or just `Enter`. Your data is saved in `~/.ironclaw/data/`.

**2. LLM provider**

```
Select inference provider:
  [1] Ollama (local) ← recommended
  [2] OpenAI-compatible
  [3] Anthropic
  ...
```

Select `Ollama`. Then enter the model name you pulled:

```
Ollama model name [llama3.2]: llama3.2
```

**3. Secrets storage**

```
Where to store the master encryption key?
  [1] System keychain ← recommended for desktop
  [2] Environment variable (IRONCLAW_MASTER_KEY)
```

Select `1` for a normal desktop. If you are running IronClaw on a headless server or in a container, select `2` and set the `IRONCLAW_MASTER_KEY` environment variable.

Setup is complete. IronClaw starts in TUI mode.

<Accordion title="⚠️ Error: Another IronClaw instance is already running">

```bash
# Remove the stale PID file and restart
rm ~/.ironclaw/ironclaw.pid
IRONCLAW_PROFILE=local ironclaw
```

</Accordion>

<Accordion title="⚠️ Error: Failed to bind to port 8080">

IronClaw's web gateway uses port 3000 in the local profile (the web gateway is disabled by default in the `local` profile). If another process is blocking port 3000:

```bash
GATEWAY_PORT=3001 IRONCLAW_PROFILE=local ironclaw
```

</Accordion>

</Step>

<Step title="Chat with your agent">

Type your first message at the prompt:

```
> What can you do?
```

IronClaw will describe its capabilities — browsing the web, searching files, taking notes, managing calendar events, and more.

Try a few tasks:

```
> Search for files named *.log in this folder
> Remind me to review my notes tomorrow at 9am
> Browse https://news.ycombinator.com and summarise the top 3 stories
```

<Note>
The browser tool requires `node` and `npx` to be installed. Run `npm install -g playwright && npx playwright install chromium` to enable it.
</Note>

</Step>

<Step title="Make the profile permanent">

So you do not need to type `IRONCLAW_PROFILE=local` every time, add it to your IronClaw environment file:

```bash
mkdir -p ~/.ironclaw
echo "IRONCLAW_PROFILE=local" >> ~/.ironclaw/.env
```

Now you can start IronClaw with just:

```bash
ironclaw
```

</Step>

<Step title="Keep IronClaw up to date">

```bash
ironclaw-update
```

Or reinstall using the same installer command. Your data and settings are never affected by updates.

</Step>

</Steps>

## Optional: Enable Semantic Memory

Semantic memory allows IronClaw to recall past conversations and notes using meaning rather than exact keywords. It requires an embedding model:

```bash
# Pull a small, fast embedding model via Ollama
ollama pull nomic-embed-text
```

Then add to `~/.ironclaw/.env`:

```env
EMBEDDING_ENABLED=true
EMBEDDING_MODEL=nomic-embed-text
```

Restart IronClaw. Semantic search is now active.

## Optional: Enable the Browser Tool

The browser tool lets IronClaw navigate websites, extract content, and take screenshots using Playwright.

```bash
# Install Node.js if not already installed: https://nodejs.org
node --version   # must be 18+

# Install Playwright and the Chromium browser
npm install -g @playwright/test
npx playwright install chromium
```

No further configuration needed — IronClaw discovers `npx` automatically.

## Optional: Connect a Calendar (CalDAV)

IronClaw supports any CalDAV server — Nextcloud, Radicale, iCloud, Fastmail, etc.

```env
CALDAV_URL=https://your-server/dav/calendars/user/
CALDAV_USERNAME=your-username
CALDAV_PASSWORD=your-password
```

Ask IronClaw to list your events or create a new one:

```
> What meetings do I have this week?
> Add a dentist appointment on Friday at 3pm
```
